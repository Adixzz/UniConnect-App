🎓 UniConnect
UniConnect is a mobile app that simplifies scheduling and communication between university students and lecturers — no more waiting on emails or guessing availability.

✨ Features
👨‍🎓 Students
View lecturer availability in real-time
Request meeting slots with a reason
Get instant notifications on request status

👩‍🏫 Lecturers
Manage all meeting requests in one place
Accept or decline requests بسهولة
Receive real-time alerts for new requests

🛠️ Tech Stack
Flutter & Dart
Firebase Firestore
Firebase Auth
Firebase Cloud Messaging (FCM)

🚀 Run Locally
flutter pub get
flutter run
